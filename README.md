# Multi-Agent Collaborative Framework for Automated Narrative and Visual Content Generation

## 🚀 Project Overview
This project is an intelligent system that automates the end-to-end process of storytelling and visual storyboarding. It uses a multi-agent orchestration architecture where specialized AI agents collaborate to generate 1000-word narratives, extract character metadata, and produce high-fidelity visual assets.

## 🛠️ Tech Stack
- **Backend:** FastAPI (Python)
- **Narrative Agent:** Llama 3.3 (via Groq)
- **Visual Agent:** FLUX.1-schnell (via Hugging Face)
- **Database:** SQLite
- **Document Engine:** FPDF2

## 🌟 Key Features
- **Semantic Synchronization:** 1000-word narratives are parsed for character traits to ensure 100% visual consistency.
- **Model-Agnostic Resilience:** Features an intelligent failover layer to mitigate API rate limits and 429 errors.
- **Automated Storyboarding:** Instantly compiles narratives and generated imagery into a professional PDF.
